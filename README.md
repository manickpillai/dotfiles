
### Install apps via homebrew

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

brew bundle install --file=Brewfile
```

To take a new backup of installed file run 

```shell
brew bundle dump
```
